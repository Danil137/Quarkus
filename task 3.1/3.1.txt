Задание:

*Создать ресурс, позволяющий выполнять операции с придуманной сущностью.
*Реализовать пример возврата кастомного HTTP кода ответа, например при ошибке удаления.
REST сервис может получать / возвращать не только JSON, необходимо реализовать это (например, вернуть строку, сделать обновление сущности по ИД через URL).
*Реализовать HTTP фильтр REST запросов и выполнить логирование информации на консоль.
Контрольные точки:

Какие форматы запроса / ответа могуть быть у REST сервисов?
Какие HTTP коды может возвращать REST сервис?
Каким образом можно перехватить возможные ошибки функционала сервиса и какой HTTP код необходимо при этом возвращать?
